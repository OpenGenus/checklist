Week 1: Getting Started

HTML is the set of markup symbols or codes inserted into a file intended for display on the Internet.
  • Introduction
    HyperText Markup Language (HTML) is the language that powers the structure of every webpage you encounter. It's like the blueprint that defines how content is presented on the internet.
  • XML
    XML stands for Extensible Markup Language. It was developed to encode data and information in a format that could be easily interpreted by humans and computers alike.
  • SHTML
    SHTML SHTML is a file extension (.shtml) that denotes that the current file is an HTML document (.html) with an extra feature known as Server Side Includes. This was a widely used feature in early 2010s.
  • HTML
    HTML and CSS are essential for web development and a good understanding of the key terms used is necessary.
  • Types of HTML
    There are 4 different types of HTML file extensions and the extension denote a specific feature enabled in the HTML web page Types of HTML.
  • Doctype
    DoCTYPE is a declaration, but definitely not a tag .

Week 2: HTML Tags

HTML tags are fundamental to web development, as they allow developers to define the structure and content of a webpage, making it readable and interactive for users.
  • Head Tag
    Head Tag in HTML contains all the header elements..
  • Body Tag
    Body Tag defines the body of the webpage.
  • Div
    The Div element is used as a container for other HTML elements.
  • Button Tag
    Button Tag Buttons are like the ears of the websites we create.When they are clicked, they listen to our options or choices and subsequently required actions take place.
  • Span Tag
    Span Tag span is similar to all the other HTML elements, it starts with an opening tag span and ends with an closing tag span.
  • Footer Tag
    Footer Tag In HTML, the footer element is used to define a footer for a section or a page. The footer typically contains metadata, copyright information, links to related documents, authorship details, or any other information that is relevant to the content enclosed by the surrounding section or the entire page.
  • Del Tag
    The Del Tag is used to display a line midway across a text.
  • Embed Tag
    The Embed Tag embed tag embeds interactive content or external application (plugin) in the specified part of HTML document.
  • Audio Tag
    Audio Tag element is used to embed sound or music element in our HTML page.
  • Video Tag
    Video Tag embeds a media player which supports video playback into the document. We can use video tag for audio content as well, but the audio tag may provide a more appropriate user experience..
  • No Script Tag
    Noscript Tag can be refer to a situation where JavaScript code is implemented or executed without using the script tag.

Week 3: Advanced HTML Tags

HTML Tags below clearly describes its meaning to both the browser and the developer.
  • Heading
    Heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading.
  • HTML Lists
    Lists Lists are used commonly to create a set of data/information having common features using bullets/numbers.
  • HTML Tables
    HTML Tables HTML tables allow developers to arrange data into rows and columns.
  • Datalist
    Datalist In HTML, the datalist element is used to provide a set of predefined options for other form-related elements, such as input and textarea. It allows you to specify a list of options that users can choose from, typically in a dropdown menu associated with an input field.
  • Canvas
    Canvas element is part of HTML5 specification and allows for the rendering of 2D shapes and bitmap images. It is a low level, procedural model that updates a bitmap.
  • Links
    Links HTML links are hyperlinks. They are used to link multiple web pages together or a part of a webpage within a webpage as a bookmark.
  • # in href
    # in href is used to indicate that the link does not have a destination URL and that its behavior is defined by JavaScript code. When a user clicks on a link with javascript:void(0) in the href attribute, nothing happens by default. However, you can use JavaScript to define an action for the link, such as displaying a popup or executing a function.
  • Superscript
    The Superscript is a text that appear above the preceding text which means it lies on half a character above the normal base line and the font is smaller.
  • Viewport
    The Viewport is the area of a web page which is visible to the user. It is not fixed in size, and varies with the size of the screen of different devices.

Week 4: HTML Attributes

HTML attributes provide additional information about HTML elements.
  • Id and Class name
    Id and Class, Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page.
  • SVG
    SVG stands for Scalable Vector Graphics and it is one of the modern ways of displaying images in websites and applications just like displaying PNG, GIF, and JPG..
  • Images
    Images can be inserted into our HTML page using image tag. The image tag is an empty tag, which means that it does not have a closing tag. We can insert parameters of the image element using various attributes like src, alt, height and width..
  • Favicon
    A favicon is a small icon displayed left of the page title in a browser tab. It is loaded directly from the page requested, and is usually a logo representing the owner of the site.
  • Fonts
    Fonts Fonts are very important as it gives a different learning experience. Today, Computers have different fonts due to the contribution of Steve Jobs which gave Apple its first advantage over competitors like Microsoft. When building a web page in HTML, we need to consider various factors such as screen size to provide the best fonts in the perfect format.
  • Radio Button
    Radio Buttons are clickable circles that are a type of an input element in HTML.
  • MetaData
    The Metadata is the data information about data. In webpages, metadata comprises of key words and phrases that describe the content of the webpage.

Week 5: HTML Features

HTML encompasses a diverse array of features that facilitate the creation of interactive and user-friendly webpages.
  • Tooltip
    Tooltip In HTML, a tooltip is a user interface element that provides additional information or context when a user hovers over or interacts with a specific element on a webpage. It typically appears as a small pop-up box containing supplementary details related to the hovered element.
  • Mathml
    Mathml is a browser feature to support rendering of mathematical equations .
  • Drag and Drop
    Drag and Drop Drag and drop is a feature that is used to move objects from one place to another. To move an object in HTML, drag it by holding on the object using left mouse click and move the mouse to the destination where it is to be dropped..
  • Pagination
    Pagination Pagination refers to the division of content into separate pages, where each page contains a fixed number of items. This is commonly used to display large sets of data such as search results, blog posts, or products in an online store.
  • Navigation bar
    Navigation bar Almost all modern websites have a nav bar of some format, and the better-looking and functional it is, the more it improves user experience and gives a great impression, especially if you want to add it to your portfolio projects.

Week 6: Advanced HTML Topics

Advanced HTML concepts encompass a diverse array of subjects distinct from fundamental HTML topics.
  • Feature Support
    Sometimes if you want to support a large variety of browsers, while still being able to use features provided by newer browsers. Other times, different browsers might provide different ways of doing the same thing. In both these cases, you will need to detect which features are supported by some browser and adapt accordingly..
  • File API
    File API HTML5 introduced the File API as part of the web platform. The File API is a set of JavaScript interfaces that allow web applications to interact with files on the user's local system.
  • XMLHttpReques
    XMLHttpReques The XMLHttpRequest (XHR) is an API in web development that provides a way to interact with servers using JavaScript. It is not specific to HTML but is often used in conjunction with HTML and JavaScript to create dynamic and interactive web pages.
  • IndexedDB
    IndexedDB IndexedDB (Indexed Database) is a web API provided by modern web browsers, including those that support HTML5. It allows web developers to create, read, write, and manage a client-side database in the user's browser.
  • WebRTC
    WebRTC WebRTC stands for Web Real-Time Communication. It is a free, open-source project that provides web browsers and mobile applications with real-time communication via application programming interfaces (APIs).
  • reCAPTCHA Verification
    reCAPTCHA Verification ReCAPTCHA verification in HTML typically refers to the integration of Google's reCAPTCHA service into a web page to add an additional layer of security..
  • Embed PDF
    For those who want a professional and beautifully designed way to display documents onto their webpages, being able to embed pdf html code onto their website would give them the ability to display interactive documents for users to use.
  • Push API
    The Push API gives web applications the ability to receive messages pushed to them from a server, whether or not the web app is in the foreground, or even currently loaded, on a user agent.
  • Cookies
    Cookies are the most commonly used technique by which websites store information on a user's browser..
  • Third Party Cookies
    Third part cookies are cookies set by external websites which we do not visit directly. A widespread use of such an external site can result in tracking and other implications.
  • HTML DOM
    HTML DOM (Document Object Model) is the structure of HTML elements that defines the way a web page is displayed on a browser. HTML DOM consists of various elements, such as headings, paragraphs, tables, and images, among others.
  • History Object
    History Object The browsers history is present in the history object. We can use the window part (window.history) or just simply history.
  • Storage Session
    Storage Session The sessionStorage property is used to access a session Storage object for the current origin URL.
  • Star Rating
    Star Rating In the digital age, user ratings have become an essential component for evaluating the quality and popularity of products, services, and content. Star ratings, in particular, have gained widespread recognition as a user-friendly and intuitive way to express opinions and provide feedback.
  • Clipboard API and Events
    Clipboard API The Clipboard API provides a way to hook into the common clipboard operations of cutting, copying and pasting so that web application can adjust the clipboard data as required.
  • WebP
    WebP The official Google page for WebP includes links to tools and libraries (Windows, MacOS, Linux) for encoding, decoding, viewing, and animating the WebP format.
  • Media Source Extension
    Media Source Extension Media Source Extensions extends HTMLMediaElement to allow JavaScript to generate media streams for playback.
  • URL Forwarding
    URL Forwarding Webpage (URL) redirection, which can also be called URL forwarding, is a popular procedure to give more than one URL address to a page, a form, or a whole Web site/application.
  • Geolocation API
    The HTML Geolocation API is used to get the current position of the user. The Geolocation API is used using the navigator.geolocation object.
